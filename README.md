# 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de centralizar minhas principais informações profissionais em uma única aplicação.

Aqui você encontrará:

- Apresentação pessoal
- Tecnologias que domino
- Projetos desenvolvidos
- Experiências
- Contato

Além de servir como meu portfólio profissional, este projeto também foi utilizado para praticar conceitos de React, componentes reutilizáveis, responsividade e boas práticas de desenvolvimento Front-end.

# 🚀 Como executar

Clone o projeto

```bash
git clone https://github.com/JoaoVitorRamos7/portfolio.git
```

Entre na pasta

```bash
cd portfolio
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```
